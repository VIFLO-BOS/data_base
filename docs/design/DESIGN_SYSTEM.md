# Design System

## Colors
- Primary: Blue (#3b82f6)
- Secondary: Slate (#64748b)
- Success: Green (#22c55e)
- Warning: Amber (#f59e0b)
- Danger: Red (#ef4444)

## Typography
- Headings: Inter, 600-700 weight
- Body: Inter, 400 weight
- Mono: JetBrains Mono (for data)

## Spacing Scale
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

## Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## Component Patterns
- Cards: rounded-lg, shadow-sm, border
- Buttons: rounded-md, transition-colors
- Inputs: rounded-md, focus-ring
- Tables: Card view on mobile, table on desktop
- Modals: Sheet on mobile, centered dialog on desktop
